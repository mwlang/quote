== Description ==

This plugin lets you add quotes from your customers and more easily manage publishing and referencing them.

== Installation ==

1. Upload plugin folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the "Plugins" menu in WordPress.
3. Go to the Quotes menu and select Add new.
4. Type in your quote.
5. Optionally enter the author's name in the "Quote Author" field.
6. Optionally enter additional info about the author or quote's source in the "Where" field.
